## Grading-System<br />
Grading System for schools, colleges and institutions<br />
Step 1<br />
```bash
git clone git@github.com:ShridharAware/Grading-System.git
```
Step 2<br />
```bash
cd Client
```
Step 3<br />
```bash
npm i --force
```
Step 4<br />
```bash
cd ..
```
Step 5<br />
```bash
cd Server
```
Step 6<br />
```bash
npm i --force
```
Step 7 <br />
To run project, navigate to particular directory and run following command <br />
```bash
npm run dev
```
